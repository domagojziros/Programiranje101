Osnovni koraci u programiranju

Varijable

Varijable su osnovni koncept u programiranju. One omogućuju spremanje podataka koje možemo koristiti kasnije.


// Primjer varijable u Pythonu:
ime = "Ana"
godine = 25
print(f"Moje ime je {ime}, a imam {godine} godina.")

Petlje

Petlje omogućuju ponavljanje blokova koda dok se određeni uvjet ispunjava.


// Primjer for petlje u Pythonu:
for i in range(5):
    print(f"Broj: {i}")

// Primjer while petlje:
brojac = 0
while brojac < 3:
    print(f"Brojac je: {brojac}")
    brojac += 1

Uvjeti

Uvjeti omogućuju donošenje odluka u programu, koristeći "if" i druge izjave.


// Primjer if uvjeta u Pythonu:
broj = 10
if broj > 0:
    print("Broj je pozitivan.")
elif broj == 0:
    print("Broj je nula.")
else:
    print("Broj je negativan.")

Funkcije

Funkcije grupiraju logiku kako bi se kod mogao ponovno koristiti i bio pregledniji.


// Definiranje funkcije:
def pozdrav(ime):
    print(f"Bok, {ime}!")

// Pozivanje funkcije:
pozdrav("Ana")

Klase

Klase omogućuju objektno orijentirano programiranje, što pojednostavljuje organizaciju koda.


// Definicija klase:
class Osoba:
    def __init__(self, ime, godine):
        self.ime = ime
        self.godine = godine

// Korištenje klase:
osoba = Osoba("Marko", 30)
print(osoba.ime, osoba.godine)
